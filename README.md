## Installation

MPFavouriteButton is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "MPFavouriteButton","~>https://github.com/MoinPansare/MPFavouriteButton.git"
```
